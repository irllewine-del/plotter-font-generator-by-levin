# Segoe Script Stroke — плоттер

Инструмент для раскладки текста на страницу и экспорта в SVG одним росчерком (single-stroke) — специально для перьевых плоттеров. Шрифт — скелетонизированный Segoe Script (607 глифов: латиница, кириллица, цифры, пунктуация, часть символов).

## Файлы

| Файл | Назначение |
|---|---|
| `plotter.html` | Сам инструмент. Полностью самодостаточный: данные шрифта зашиты внутри (`<script id="fontdata">`), открывается в браузере без сервера и без интернета. |
| `app.py` | Обёртка на [pywebview](https://pywebview.flowrl.com/), открывает `plotter.html` в нативном окне (системный WebView2). Из неё собирается `.exe`. |
| `dist/PlotterFont.exe` | Собранный standalone-exe (см. «Сборка exe» ниже). Можно распространять как один файл, Python на целевой машине не нужен. |
| `SegoeScriptStroke.ttf`, `strokes.json`, `specimen.svg` | Сопутствующие артефакты генерации шрифта. **Не используются** `plotter.html` напрямую — тот берёт данные из встроенного JSON. `specimen.svg` — сетка всех глифов для визуальной проверки, `strokes.json` — те же данные шрифта отдельным файлом (на случай, если понадобится перегенерировать `plotter.html`). |

## Использование

**Вариант 1 — просто открыть в браузере:** дважды кликнуть `plotter.html`.

**Вариант 2 — desktop-приложение:** запустить `dist/PlotterFont.exe`.

В интерфейсе:
- **Лист** — размер страницы (мм) и поля, готовые пресеты (A4, A5, A3, Letter, A6, 10×15).
- **Текст** — кегль, межстрочный интервал, выравнивание, авто-перенос строк.
- **Перо** — толщина и цвет линии, переключатель **«Сглаживание»** (сплайн Catmull-Rom поверх опорных точек штриха — убирает гранёность на кривых, включён по умолчанию).
- **Содержимое** — сам текст.
- Кнопка **«Скачать SVG»** экспортирует результат в мм-точном SVG (`viewBox` и `width/height` в миллиметрах) — можно сразу открывать в любом плоттерном ПО (Inkscape + AxiDraw extension, vpype и т.п.).
- Внизу — статистика: число полилиний, точек, длина пера и сколько строк текста поместилось на странице.

## Сборка exe заново

Если правите `plotter.html` и нужно пересобрать `.exe`:

```
pip install pywebview pyinstaller
cd plotter_font_extract
python -m PyInstaller --noconfirm --onefile --windowed --add-data "plotter.html;." --name "PlotterFont" app.py
```

Результат — `dist/PlotterFont.exe`. Промежуточные файлы (`build/`, `PlotterFont.spec`) можно удалять и пересоздавать безболезненно.

## Известные ограничения

- **Нет оптимизации хода пера.** Штрихи рисуются в порядке символов исходного текста, без учёта холостых перемещений между ними — для длинных текстов это может занимать больше времени плоттера, чем сама отрисовка.
- **Переполнение страницы малозаметно.** Если текст не помещается (слишком крупный кегль или длинный текст), лишние строки просто не рисуются — единственный признак этого — надпись «⚠ обрезано» в статистике, видимого предупреждения на самом листе нет.
- **Составные символы** (©, ®, некоторые диакритические знаки) при сведении к одной линии превращаются в малочитаемые росчерки — структурное ограничение скелетонизации многоконтурных фигур.
